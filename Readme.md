## Project Overview

This project demonstrates a basic Computer Vision pipeline using Convolutional Neural Networks (CNNs) for image classification.

The project covers:

* Image preprocessing
* Dataset analysis
* CNN architecture design
* Model training
* Performance evaluation
* Prediction visualization
